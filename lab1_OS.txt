ls - Показує список файлів або папок в самій директорії
pwd - Working directory. Ця команда показує поточне находження в директорії
cd - Змінити директорію
touch - Створює файл
cat - Показує місткість файлу
mkdir - Створити папку
echo "example" > example.txt - За допомогою цієї команди кинув текст "example" в текстовий файл example.txt
rm - Видалити файл.
mkdir dicetory - Створило директорію directory
nano - Текстовий редактор
